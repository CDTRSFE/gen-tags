# Gen Tags

用于快速生成和推送 tag。

![view](src/assets/view.jpg)

## Features

- 按规则生成 tag
- 支持前缀和后缀
- 将生成的 tag 存在 package.json 中

> 前缀通过 package.json 中的 `tagPrefix` 字段配置，如： `tagPrefix: ["v-"]`。

## Requirements

- vscode.git 扩展
